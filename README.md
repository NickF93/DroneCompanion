# DroneCompanion

DroneCompanion is an Unreal Engine 5.5 C++ project built around a small runtime plugin for a companion drone. The plugin is designed for a Game Programming course blockout: it keeps the gameplay code in C++, uses Blueprint only for editor-facing setup, and demonstrates follow movement, target sensing, collectible inspection, enemy attack, and collision-aware pawn movement.
